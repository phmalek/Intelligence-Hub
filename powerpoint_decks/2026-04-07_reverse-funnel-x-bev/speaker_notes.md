# Speaker Notes

## Slide 1

`Reverse Funnel x BEV.`

`This is a shift from media planning to outcome engineering.`

`We do not start with budget.`

`We start with the sales target.`

`From the sales target, we calculate the required lead volume.`

`From required leads, we calculate the sessions needed to generate them.`

`From sessions, we calculate the media spend required to create that traffic.`

`Only then do we decide channel allocation.`

`That is the reverse funnel.`

`The system is built on observed performance, not assumptions.`

`So if conversion changes, lead need changes.`

`If lead need changes, session need changes.`

`If session need changes, spend and allocation change with it.`

`That gives us a closed loop, not a static plan.`

`The allocation layer then shifts budget across search, inventory, and social based on marginal return.`

`Where a channel still has headroom, we scale it.`

`Where a channel is saturating, we pull back.`

`So every euro can be traced back to sales impact.`

## Slide 2

`This slide shows the operating logic behind the system.`

`First, start from sales.`

`Budget is derived from targets, not guesses.`

`Second, find the gap.`

`We quantify where the funnel is underperforming.`

`Third, reallocate.`

`We move spend to the highest marginal return.`

`Fourth, scale the system.`

`What works is codified and transferred across markets.`

`And fifth, the output is not a media plan.`

`The output is an always-on decision engine.`

`On the right, marginal return declines as spend increases.`

`That tells us where each extra euro is still productive, and where it is starting to lose efficiency.`

`Headroom is the remaining spend capacity before a channel reaches saturation.`

`So the practical decision is simple.`

`Invest where there is still efficient headroom.`

`Reduce pressure where the curve is flattening.`

`That is how reverse funneling turns demand into a controllable commercial system.`
